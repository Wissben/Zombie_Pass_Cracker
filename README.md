Zombie_Pass_Cracker
=================

This is a distributed password cracker for linux system.

supported hashes:
* MD5
* SHA512

How to run
------------

1) Run the python server **server.py**.

![image0_server](images/images0.png)
![image1_server](images/images1.png)

2) run **make.sh** to compile the client.
